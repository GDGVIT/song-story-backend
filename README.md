## Instructions to run

* Pre-requisites:
	- System with atleast 4GB RAM
	- Download the trained models from Google Drive

* Directions To Install

```bash
cd to/project/directory 
pip install -r requirements.txt
```

* Directions To Execute

```bash
cd to/project/directory
python3 song_story/app/run.py
```
